# dotproduct.one - Landing Page

Landing page with waitlist for dotproduct.one

## Local Development

```bash
# Install dependencies
npm install

# Run dev server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Deploy to Netlify

1. Push this repo to GitHub
2. Connect to Netlify
3. Netlify will auto-detect Next.js and deploy
4. Waitlist emails will be collected in Netlify Forms dashboard

## Waitlist Data

View submitted emails in Netlify dashboard under Forms section.
